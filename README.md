## 环境
vue -v2.6.14  
[vux -2.11.1](https://github.com/airyland/vux)  
element-ui -v2.15.9  
vue/cli -v5.0.0  
webpack -v5.0.0  
node -v14.20.0

所有围绕[vux](https://github.com/airyland/vux)工具都已经升级到最新最稳定版本
